# bitmesh

## Tests on Docker network
See the directory test. Or,

To build docker images,
```
make
```

To run first chord node,
```
make first
```

To run more chord nodes,
```
make node
```

To run dht test,
```
make dht
```

To run node caller test,
```
make caller
```

To stop all containers,
```
make stop
```

To clean up containers,
```
make clean
```
